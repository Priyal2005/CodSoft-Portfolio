My Skills

C++ is a programming language developed by Bjarne Stroustrup in 1985. It combines the efficiency and performance of C with modern features like classes and objects, enabling robust software development.

HTML (HyperText Markup Language) is the standard language for creating web pages. It’s the foundation of web development, allowing developers to structure content and elements on the internet.

CSS (Cascading Style Sheets) is the language used to style and layout web pages. While HTML structures a webpage's content, CSS makes it visually appealing.

JavaScript is used to enhance HTML and CSS by adding interactive elements such as animations, form validation, and dynamic content updates without reloading the entire page. It works across all major web browsers and integrates seamlessly with other web technologies.

Git is a distributed version control system that helps developers manage changes to source code over time. It’s incredibly useful for collaboration and keeping track of different versions of a project.

GitHub is a web-based platform that uses Git for version control and collaboration. It allows developers to store and manage their code, track changes, and collaborate on projects.

Responsive design is a web design approach aimed at crafting websites that provide optimal viewing and interaction experiences across a wide range of devices (from desktop computers to mobile phones).
